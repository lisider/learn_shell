
- 正则
```c
一种语法,是为了写表达式
一个正则表达式 是一个 字符串,
	字符串里面的全部字符(物理上)包括
		字符 
		元字符 
		'/'
	而逻辑上包括
		一个字符集 
		锚
		修饰符
	
而 正则语法 分为
	标准
	扩展(比 标准 多几个元字符)

```

- 正则的目的
```c
过滤/转换  IO/文件 流

```

- 正则的位置
```c

命令的参数
使用正则表达式 的不是 shell  是 其他的命令,像 grep awk sed 

注意: 键入的正则表达式不要被shell解析,要注意用 全引用 '' ,或者转义 \

```
