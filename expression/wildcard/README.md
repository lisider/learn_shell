- 通配

```c

展开文件名扩展
也是一个字符串
          字符串在物理上包括
		*
		?
		[]
		^
          专用字符集合

               [:digit:]：任意数字，相当于0-9
               
               [:lower:]：任意小写字母
               
               [:upper:]：任意大写字母
               
               [:alpha:]：任意大小写字母
               
               [:alnum:]：任意数字或字母
               
               [:space:]：空格
               
               [:punct:]：标点符号

               引用时，需要再加一层括号
```

- 通配的目的
```c

展开文件名扩展

```

- 正则的位置

```c

命令的参数

使用通配的 是shell ,不是 其他的命令

注意 : 键入的通配 一定要被shell 解析

```
